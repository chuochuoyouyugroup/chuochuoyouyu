# 数据库设计
# 1.引言
## 1.1 编写目的
本文档主要对在线英语中文交互学习系统在设计阶段对数据库设计的说明和描述。本文档的主要读者应为后续设计阶段的设计人员、实现阶段的编码人员和实际使用时的系统管理员。
## 1.2 背景
本次需要开发的系统以“在线英语中文交互学习系统”作为本次系统的命名，最终的产品是生产出符合本需求文档的英语在线学习系统。
